
# AI素养操作化敏感性分析

> **目的**：检验"AI素养"不同操作化方式对组别效应估计的稳健性。
> 三种模型均包含 `group_c` 和 `self_assessed_ability`（前测自评能力）作为基础控制变量，
> 在此基础上分别加入不同的AI素养测量。

| 模型标签 | AI素养操作化 | 公式 |
|---------|------------|------|
| 模型A | 仅AI使用频率 | `group_c + ai_exposure_num + self_assessed_ability` |
| 模型B | 仅AI熟悉度  | `group_c + ai_familiarity + self_assessed_ability` |
| 模型C | 综合分（z均值） | `group_c + ai_literacy_composite + self_assessed_ability` |

> **综合分**：`ai_literacy_composite = (z(ai_familiarity) + z(ai_exposure_num)) / 2`
> 两变量原始相关：r = 0.297
