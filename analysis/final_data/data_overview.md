## 一、数据结构

### 1.1 样本过滤流程

| 步骤                          | 操作                       | 保留 n                            |
| --------------------------- | ------------------------ | ------------------------------- |
| 完成全部 21 张图像                 | —                        | 119 |
| 通过注意力检测                     | 排除 5 人                   | 114                             |
| 手动质检排除                      | delete = 1，共 3 人         | 112                             |
| **Manipulation Check（C 组）** | C 组中未读干预材料或未读策略列表，排除 6 人 | **106**                         |

### 1.2 核心数据文件（final_data/）

| 文件                     | 内容                                        | 行数（MC 过滤后） |
| ---------------------- | ----------------------------------------- | ---------- |
| `participants.csv`     | 每位被试的汇总数据（含编码 + mc_passed）                | 106        |
| `responses.csv`        | 逐题作答记录（is_correct, confidence, judgment…） | 2283       |
| `interaction_logs.csv` | 行为日志（OPEN_LENS, SCROLL_LENS…）             | 571        |
| `post_survey.csv`      | 后测问卷（manipulation check, strategy_usage…） | 106        |

### 1.3 核心变量说明

| 变量                        | 操作化                                      | 量程  | 来源           |
| ------------------------- | ---------------------------------------- | --- | ------------ |
| `acc_total`               | 正确判断数 / 21                               | 0–1 | responses    |
| `acc_ai`                  | AI 图正确率                                  | 0–1 | responses    |
| `acc_real`                | 真实图正确率                                   | 0–1 | responses    |
| `d'`（dprime）              | SDT 敏感度，z(HR) − z(FAR)，Loglinear 校正      | 连续  | 计算           |
| `c`（criterion）            | 判断标准偏向，负=偏判为AI                           | 连续  | 计算           |
| `self_assessed_ability`   | 前测自评辨别能力                                 | 1–5 | participants |
| `self_performance`        | 后测对自身表现的整体评估                             | 1–5 | post-survey  |
| `calibration_gap`         | self_performance/5 − acc_total（正 = 过度自信） | 连续  | 计算           |
| `ai_familiarity`          | AI 工具熟悉程度                                | 1–5 | participants |
| `ai_exposure_num`         | AI 使用频率（never=1…very-often=5）            | 1–5 | participants |
| `intervention_duration_s` | 干预页面停留时长（秒，A 组填 0）                       | 秒   | participants |
