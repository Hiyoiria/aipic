Study 2 图表规格说明（用于指导绘图代码）
========================



【最高要求，可覆盖下面冲突的】不要在图内写标题、非专有词都用中文，基本只用黑白灰



> 共建议 7 张图（F1–F7），按论文章节顺序排列。配色方案统一：对照组(A) = #7FB3D8（浅蓝），干预组(C) = #F4A261（橙色）。全部使用 matplotlib + seaborn，中文字体设置 SimHei/Microsoft YaHei，英文 Arial。输出格式：300dpi PNG + 矢量 PDF。图表尺寸建议宽 6–8 inch，高 4–5 inch。

F1：干预主效应对比图
-----------

**插入位置**：4.1.1 主效应，紧接表4-1之后

**图表类型**：分组柱状图（grouped bar chart），2组 × 5指标

**数据**：

| 指标  | A组 M  | A组 SE     | C组 M   | C组 SE     |
| --- | ----- | --------- | ------ | --------- |
| 准确率 | 0.594 | 0.156/√54 | 0.658  | 0.150/√58 |
| d'  | 0.484 | 0.829/√54 | 0.835  | 0.842/√58 |
| c   | 0.041 | 0.354/√54 | -0.011 | 0.405/√58 |
| HR  | 0.572 | 0.186/√54 | 0.647  | 0.196/√58 |
| FAR | 0.405 | 0.185/√54 | 0.357  | 0.185/√58 |

**布局要求**：

* X轴：5个指标名称（中文标签：辨别准确率、敏感度d'、判断标准c、命中率HR、虚报率FAR）
* Y轴：数值（注意d'和c的量程与其他不同，考虑用双Y轴或分两个子图）
* 每个指标两根柱子（A蓝/C橙），柱顶加误差线（±1 SE）
* 显著性标注：在准确率、d'、HR柱对上方加星号 * (p<.05)
* **建议改为两个子图**：
  * 左子图(a)：准确率 + HR + FAR（量程都在0–1）
  * 右子图(b)：d' + c（量程可能在-1到+3）
  * 共享图例

**替代方案**：如果觉得5个指标太多，精简为只画准确率+d'两个核心指标即可（2组×2指标），更简洁。

F2：过度怀疑检验交互效应图
--------------

**插入位置**：4.1.3 干预未引发过度怀疑，紧接表4-4之后

**图表类型**：交互效应折线图（interaction plot）

**数据**：

| 图像类型   | A组 M(SE)          | C组 M(SE)          |
| ------ | ----------------- | ----------------- |
| AI图像   | 0.580 (0.207/√54) | 0.663 (0.218/√58) |
| Real图像 | 0.603 (0.201/√54) | 0.655 (0.200/√58) |

**布局要求**：

* X轴：图像类型（AI图像 / 真实图像），仅两个刻度
* Y轴：正确率（0.4–0.8范围，不从0开始以放大差异）
* 两条折线：A组（蓝，圆形marker）、C组（橙，方形marker）
* 每个点加误差线（±1 SE）
* 在AI图像的两点之间标注差异值和显著性（Δ=+8.3%, p=.042*）
* 在Real图像的两点之间标注（Δ=+5.2%, p=.175 n.s.）
* 关键视觉信息：两条线近似平行（交互不显著），C组始终高于A组，差距在AI图上稍大

**设计要点**：

* 线条粗细区分（C组线稍粗或实线，A组虚线）
* 背景网格浅灰色
* 图例放右上角

F3：简单斜率分析/调节效应图
---------------

**插入位置**：4.3.2 前测自评能力的调节效应，紧接表4-6之后

**图表类型**：调节效应散点+回归线图

**数据来源**：需要被试级原始数据（pre_self_ability, accuracy, group）

**布局要求**：

* X轴：前测辨别能力自评（1–5，中心化后或原始值均可）
* Y轴：辨别准确率（0–1）
* 两条回归线：A组（蓝虚线）、C组（橙实线），带95%置信区间阴影带
* 散点：A组蓝色半透明圆点，C组橙色半透明方点（alpha=0.3避免重叠）
* 标注简单斜率结果：
  * 在X轴低端（≈2）附近标注"低自评: B=0.092, p=.015*"
  * 在X轴高端（≈4）附近标注"高自评: B=0.016, p=.680"
* 关键视觉信息：两条回归线在低自评端差距大，高自评端趋于收敛

**设计要点**：

* 散点用jitter避免整数值重叠（自评是1-5整数）
* 置信区间阴影带透明度0.15
* 如果散点太密，可改用分组均值+误差线的简化版（在自评=1,2,3,4,5五个点上画两组均值）

**简化替代方案**：

* X轴：三个水平（低自评-1SD / 均值 / 高自评+1SD）
* Y轴：干预效应B值
* 三个点+误差线（95%CI），标注显著性
* 类似森林图的横向版

F4：异质性分析森林图
-----------

**插入位置**：4.3.3 异质性分析，紧接表4-7之后

**图表类型**：森林图（forest plot）

**数据**：

| 子群        | B     | SE    | 95% CI下限 | 95% CI上限 | p      |
| --------- | ----- | ----- | -------- | -------- | ------ |
| 全样本       | 0.065 | 0.029 | 0.008    | 0.122    | .027*  |
| 性别：男      | 0.108 | 0.040 | 0.030    | 0.186    | .009** |
| 性别：女      | 0.044 | 0.041 | -0.037   | 0.125    | .289   |
| 年龄：≤34    | 0.059 | 0.033 | -0.006   | 0.124    | .075   |
| 年龄：≥35    | 0.084 | 0.064 | -0.052   | 0.220    | .201   |
| AI频率：低    | 0.018 | 0.036 | -0.054   | 0.090    | .622   |
| AI频率：高    | 0.112 | 0.041 | 0.031    | 0.193    | .008** |
| 自评：低(1-2) | 0.084 | 0.049 | -0.017   | 0.185    | .097   |
| 自评：高(3-5) | 0.040 | 0.034 | -0.028   | 0.108    | .246   |

> 注：SE为近似值，精确值需从原始回归输出中提取。CI = B ± 1.96×SE。

**布局要求**：

* 纵向排列，每行一个子群
* 左侧：子群名称（中文）
* 中间：效应量点估计（菱形/圆形marker） + 95% CI 横线
* 右侧：B值 [95% CI] 数值标注
* 全样本行加粗或用菱形标记区分
* 垂直参考线：x=0（虚线，零效应）
* 可选：x=0.065（全样本效应，灰色虚线）
* 用灰色条纹交替区分不同维度（性别区/年龄区/AI频率区/自评区）
* 显著的子群（p<.05）用实心marker，不显著的用空心
* 在右侧额外加一列标注Chow检验p值

**设计要点**：

* 最上方一行是"全样本"（总效应参考）
* 然后按维度分组，每组之间加小间距
* CI横线如果跨越0则用灰色，不跨越0用黑色/彩色
* 图宽建议8-9 inch以容纳右侧数值

**布局要求**：

* Y轴：21张图片ID（按Δ降序从上到下），ID旁标注图片类型标签
  * AI图用 ■ 红色/深色小方块前缀
  * Real图用 ● 绿色/浅色小圆点前缀
* X轴：正确率（0–1）
* 每张图两根水平条形（A蓝/C橙），并排或重叠
* 在Δ>0.15的图对旁标注差异值
* 50%基准线（竖直虚线，表示随机猜测水平）
* real_11旁标注"原始p=.040*"（唯一未校正显著的）

**替代方案（更简洁）**：

* 只画Δ(C-A)的瀑布图/哑铃图（dumbbell chart）
  * 每行一张图，两个点（A和C），用线段连接
  * 线段颜色：C>A为橙色，A>C为蓝色
  * 更清晰地展示差异方向和幅度

**设计要点**：

* 图高需要足够（至少7-8 inch）以容纳21行
* 字体不宜过小（≥9pt）
* 可在图底部加注释："虚线表示随机猜测水平（50%）"

F6：校准曲线图
--------

**插入位置**：4.2 策略干预对辨别信心的影响（作为补充分析）

> 需要逐图信心数据（confidence 0-5）。如果决定在正文使用，放在4.2.2之前；如果篇幅有限，放附录。

**图表类型**：折线图（calibration curve）

**数据**：需从 responses_combined.csv 计算：

* 对每个信心等级(0,1,2,3,4,5)，分A/C组计算该信心等级下的实际正确率
* 同时统计各信心等级的作答次数n（用于判断数据量是否足够）

| 信心等级 | A组正确率 | A组n | C组正确率 | C组n |
| ---- | ----- | --- | ----- | --- |
| 0    | [需计算] |     | [需计算] |     |
| 1    | [需计算] |     | [需计算] |     |
| 2    | [需计算] |     | [需计算] |     |
| 3    | [需计算] |     | [需计算] |     |
| 4    | [需计算] |     | [需计算] |     |
| 5    | [需计算] |     | [需计算] |     |

预实验参考值（全样本）：

* 信心0: acc=0.561, n=651
* 信心1: acc=0.339, n=180
* 信心2: acc=0.521, n=259
* 信心3: acc=0.584, n=413
* 信心4: acc=0.731, n=383
* 信心5: acc=0.849, n=503

**布局要求**：

* X轴：信心等级（0–5）
* Y轴：实际正确率（0–1）
* 两条折线：A组（蓝，圆marker）、C组（橙，方marker）
* 对角线参考线（灰色虚线）：完美校准线（信心/5 = 正确率）
  * 即(0, 0), (1, 0.2), (2, 0.4), (3, 0.6), (4, 0.8), (5, 1.0)
* 每个点旁标注n值（小字体），数据量太少的点(<30)标为空心
* 50%水平参考线（浅灰虚线，随机猜测基准）

**关键视觉信息预期**：

* 低信心(0-1)时正确率可能反而较高（因为信心0可能包含"完全不知道所以随便选"的人）
* 高信心(4-5)时两组可能出现分化——如果C组的高信心正确率更高，说明干预改善了"信心质量"
* 整体曲线应在对角线下方（信心普遍低于实际表现，即过度保守），与预实验一致

**设计要点**：

* 可在marker大小上映射n值（n越大marker越大）
* 如果信心=0的数据很多且pattern异常，可加注释说明

F7：相关矩阵热力图
----------

**插入位置**：4.6 相关性分析

**图表类型**：下三角热力图（lower triangle heatmap）

**数据**：

|       | 准确率   | d'    | 前测自评  | AI熟悉度 | AI频率  | 校准差距 |
| ----- | ----- | ----- | ----- | ----- | ----- | ---- |
| 准确率   | 1     |       |       |       |       |      |
| d'    | .986  | 1     |       |       |       |      |
| 前测自评  | .384  | .384  | 1     |       |       |      |
| AI熟悉度 | .285  | .274  | .542  | 1     |       |      |
| AI频率  | .260  | .271  | .282  | .241  | 1     |      |
| 校准差距  | -.546 | -.516 | -.086 | -.073 | -.127 | 1    |

**布局要求**：

* 只显示下三角（对角线上方留白）
* 色阶：红-白-蓝（diverging colormap，红=正相关，蓝=负相关，中心白=0）
  * 或 RdBu_r / coolwarm
* 每个格子中标注相关系数值（保留2-3位小数）
* 显著性标注：* p<.05, ** p<.01, *** p<.001（叠加在系数值后面）
* 对角线格子可用灰色填充或留空
* 变量标签用中文：辨别准确率、敏感度d'、前测自评能力、AI熟悉度、AI使用频率、信心校准差距
* 色阶条（colorbar）放在右侧，范围 -1 到 +1

**设计要点**：

* 格子大小均匀，字体≥10pt
* 强相关（|r|>.5）的格子颜色饱和度高，弱相关的接近白色
* 可选：在格子边框上区分p<.05（细边框）和p<.01（粗边框）
