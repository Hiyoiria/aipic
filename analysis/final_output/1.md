研究二　结果
======

一、数据与样本
-------

本研究共招募参与者163人。经两轮筛选后，最终保留有效样本152人（对照组78人，实验组74人）。第一轮依据注意力检验题排除5人；第二轮对实验组参与者进行干预理解核查（manipulation check），排除未通过者6人。最终样本规模满足事前G*Power 3.1功效分析所估算的最低需求（N = 128）。实验刺激材料共使用21张有效图像（9张AI生成图像，12张真实图像），涵盖摄影写实、插画及卡通三种风格。

* * *

二、基线等价性检验
---------

随机分组的有效性有赖于两组在基线特征上的可比性。为此，本研究在实验开始前对人口统计学变量和AI素养相关指标分别进行了等价性检验。

在人口统计学方面，卡方检验结果显示，两组在性别（χ²(1) = 2.60, _p_ = .107, _V_ = .13）、年龄（χ²(3) = 4.69, _p_ = .196, _V_ = .18）和教育程度（χ²(2) = 0.86, _p_ = .651, _V_ = .08）上均无显著差异，效应量均处于微小水平。在AI素养基线指标方面，Welch's _t_检验结果亦表明，两组在前测自评辨别能力（对照组 _M_ = 2.90, _SD_ = 1.18；实验组 _M_ = 3.09, _SD_ = 1.12；_t_(150.0) = 1.055, _p_ = .293, _g_ = .170）和AI使用频率（对照组 _M_ = 3.50, _SD_ = 1.11；实验组 _M_ = 3.53, _SD_ = 1.10；_t_(149.7) = 0.150, _p_ = .881, _g_ = .024）上均无显著差异。上述结果表明，随机分配过程成功实现了两组在主要人口统计学特征与AI认知基线上的初始等价性，为后续因果推断提供了有效保障。

* * *

三、干预主效应
-------

### 3.1　组间均值比较

为检验策略干预对AI图像识别能力的影响，本研究首先通过Welch's _t_检验对两组在多项核心指标上的均值差异进行比较（见表3）。

在整体正确率上，实验组（_M_ = 0.676, _SD_ = 0.154）显著高于对照组（_M_ = 0.601, _SD_ = 0.150），_t_(149.1) = 3.012, _p_ = .003, _g_ = .487，效应量达到中等水平。在信号检测理论（SDT）框架下，以敏感度指标*d'*衡量参与者区分AI图像与真实图像的辨别能力，实验组（_M_ = 0.933, _SD_ = 0.878）同样显著优于对照组（_M_ = 0.524, _SD_ = 0.787），_t_(146.2) = 3.023, _p_ = .003, _g_ = .490，与正确率指标高度吻合。

在具体操作指标上，实验组的命中率（HR，正确识别AI图像的比例）显著高于对照组（0.659 vs. 0.588，_p_ = .016, _g_ = .394），虚报率（FAR，将真实图像误判为AI的比例）则显著低于对照组（0.339 vs. 0.404，_p_ = .027, _g_ = .360）。这一模式表明，干预在提升参与者识别AI图像能力的同时，也有效降低了其对真实图像的误判倾向，干预效果呈现为整体辨别力的全面提升，而非单一方向的偏移。

与此形成对照的是，两组在判断标准_c_上并无显著差异（对照组 _M_ = 0.019；实验组 _M_ = −0.001；_p_ = .720），这意味着干预并未系统性地改变参与者的判断倾向或反应偏向，仅提升了其感知辨别力。

### 3.2　回归分析：控制人口统计学变量

为排除人口统计学因素的混淆影响，以性别、年龄段和教育程度（三分类：高中/大专、本科、硕博）为控制变量，分别对整体正确率和敏感度*d'*进行线性回归分析（模型一，参照组：男性、18–24岁、本科）。

在整体正确率模型中，干预变量的回归系数为 _B_ = 0.076（_SE_ = 0.025, β = .243, _t_ = 2.983, _p_ = .003），在控制人口统计学变量后仍具有显著预测力。模型整体解释量 _R²_ = .101（调整后 _R²_ = .058，_F_(7, 144) = 2.318, _p_ = .029）。在敏感度*d'*模型中，干预效应同样稳健（_B_ = 0.404, β = .237, _p_ = .004），模型 _R²_ = .102（_F_(7, 144) = 2.347, _p_ = .027）。所有人口统计学变量的回归系数均未达到显著水平，VIF值均低于1.40，不存在多重共线性问题。

### 3.3　回归分析：控制AI素养相关变量

以AI使用频率和前测自评辨别能力（均已中心化）为控制变量，建立模型二。干预对整体正确率的效应依然显著（_B_ = 0.069, β = .223, _p_ = .005），对_d'_的效应亦然（_B_ = 0.382, β = .224, _p_ = .004）。此外，前测自评能力对整体正确率（β = .160, _p_ = .044）和_d'_（β = .168, _p_ = .034）均具有独立的正向预测效果，表明认知基线较高的参与者辨别能力相对更强；而AI使用频率的预测效果则处于统计临界水平（_p_ = .058/.054），未达显著。模型二的解释量（_R²_ = .114, _F_(3, 148) = 6.356, _p_ < .001）优于模型一，表明AI素养指标对辨别能力具有一定增量效度。

综合两个回归模型的结果，策略干预对识别准确率和感知敏感度的正向效应在控制多组协变量后均保持稳健，不受人口统计学或AI素养背景因素的实质性混淆。

四、过度怀疑检验
--------

干预是否导致参与者对所有图像（包括真实图像）均产生过度怀疑，是评估干预效果质量的重要维度。为此，本研究对图像类型（AI vs. 真实）与组别的交互效应进行了混合方差分析（Mixed ANOVA）。结果显示，组别主效应显著（_F_(1, 150) = 9.328, _p_ = .003, η² = .059），图像类型主效应不显著（_F_(1, 150) = 0.004, _p_ = .947），二者交互效应亦不显著（_F_(1, 150) = 0.038, _p_ = .846）。

进一步的简单效应分析表明，实验组对AI图像的识别准确率（_M_ = 0.677, _SD_ = 0.208）显著高于对照组（_M_ = 0.598, _SD_ = 0.191），_t_(147.1) = 2.433, _p_ = .016, _g_ = .394；与此同时，实验组对真实图像的正确率（_M_ = 0.675, _SD_ = 0.199）同样显著高于对照组（_M_ = 0.604, _SD_ = 0.193），_t_(149.1) = 2.228, _p_ = .027, _g_ = .360。两类图像上组间效应量基本相当，不存在实验组仅对AI图像准确率提升、而对真实图像误判增加的差异性模式。上述结果排除了干预引发系统性过度怀疑的可能性，提示策略训练提升的是参与者的整体辨别敏感度，而非单纯催生悲观性判断偏向。

五、图像层面分析
--------

### 5.1　逐图差异

以每张图像为单位进行Fisher精确检验（组别 × 是否正确），共21次检验（Bonferroni校正后α = 0.0024）。在未校正水平上，有3张图像达到显著差异（ai_08: _p_ = .034；real_02: _p_ = .011；real_11: _p_ = .044），方向均为实验组正确率更高，差值范围为+0.157至+0.183。Bonferroni校正后，无图像达到校正显著性阈值，这与本研究样本量对单图层面的统计功效有限一致。

总体而言，21张图像中有17张呈现实验组优于对照组的正向趋势，仅4张出现小幅负差值（差值范围−0.067至−0.010），方向一致性较高，进一步佐证了干预的整体有效性。

### 5.2　图像风格的调节作用

将图像风格二分为摄影写实（photograph）与图绘类（not_photo，含插画与卡通），对两类风格的组间差异分别进行检验，并建立风格 × 组别交互回归模型（n = 304行，每人2个风格均值）。

结果显示，干预在摄影写实风格图像上的效果更为明显（_g_ = 0.483, _p_ = .003），在图绘类风格上的效果也显著但相对较弱（_g_ = 0.336, _p_ = .039）。然而，回归模型中组别与风格的交互项并不显著（_B_ = 0.020, _p_ = .638），表明两类风格之间的效果差异未达到统计意义上的调节阈值。可能的解释在于，本研究的策略教学内容以解剖结构细节和图像物理属性为核心，这些线索在摄影写实类图像中更为明显，因而产生了效果上的相对差异，但该差异尚需在更大样本中进一步检验。

### 5.3　可反向搜索性的影响

对于可通过以图搜图（reverse image search）在网络上查找到的图像，两组正确率差异未达显著（对照组 _M_ = 0.613，实验组 _M_ = 0.664；_p_ = .081, _g_ = .286），这可能反映了部分参与者通过外部知识渠道弥补了辨别能力的不足。相比之下，对于不可反向搜索的图像（均为AI生成图），干预效果显著增强（对照组 _M_ = 0.593，实验组 _M_ = 0.685；_t_(149.9) = 3.239, _p_ = .001, _g_ = .522），效应量达到中等偏上水平。该发现表明，策略教学在参与者无法借助外部信息源的情境下具有更强的实际价值，进一步凸显了内化辨别策略的必要性。

* * *

六、AI素养的调节效应
-----------

### 6.1　相关分析

在进行调节效应检验前，先行考察AI素养相关指标与辨别准确率的双变量相关。结果显示，前测自评辨别能力与准确率呈显著正相关（_r_ = .207, _p_ = .010），AI使用频率亦与准确率显著正相关（_r_ = .183, _p_ = .024），提示AI认知素养基线较高的参与者整体辨别表现更好。

### 6.2　前测自评能力的调节作用

为检验前测自评能力是否调节干预效果，以组别（中心化）、自评能力（中心化）及其交互项为核心预测变量，同时纳入性别、年龄、教育程度和AI使用频率作为控制变量，建立完整调节效应模型（模型 I）。

结果显示，干预主效应显著（_B_ = 0.072, _p_ = .004），自评能力主效应亦显著（_B_ = 0.046, _p_ = .002），且两者交互项达到显著水平（_B_ = −0.046, _p_ = .031），表明前测自评能力对干预效果具有负向调节作用。模型整体解释量 _R²_ = .185（调整后 _R²_ = .128, _F_(10, 141) = 3.209, _p_ < .001）。

简单斜率分析进一步揭示了调节效应的具体模式：在低自评能力水平（−1 SD，SAE ≈ 1.84）的参与者中，干预效应最强（_B_ = 0.125, _p_ < .001）；在均值水平（SAE ≈ 2.99）时，干预效应仍显著（_B_ = 0.072, _p_ = .004）；而在高自评能力水平（+1 SD，SAE ≈ 4.15）时，干预效应降至不显著（_B_ = 0.018, _p_ = .594）。Johnson-Neyman分析进一步确认，干预效应仅在原始自评能力量表得分低于3.50的参与者中稳定达到显著水平（_p_ < .05），当自评能力超过该阈值后干预效果逐渐消减。

上述结果具有重要的理论与实践意涵：在基线辨别能力较低的参与者中，策略训练提供了较为明显的认知"脚手架"效果；而基线能力较强者受益空间相对有限，可能存在天花板效应（ceiling effect）或其已通过其他非正式渠道习得了类似策略。

### 6.3　AI使用频率的调节作用

采用相同结构建立AI使用频率调节模型（模型 I），结果显示，组别 × AI使用频率交互项不显著（_B_ = 0.020, _p_ = .369），表明AI使用频率对干预效果不具有显著调节作用。干预主效应在控制AI使用频率后依然稳健（_B_ = 0.069, _p_ = .006），前测自评能力作为协变量保持显著（_p_ = .020）。这一结果提示，单纯接触或使用AI工具的频次并不足以替代系统性策略训练的效果，AI使用经验与策略教学在提升辨别能力方面属于相对独立的作用路径。

* * *

七、综合小结
------

综合以上分析，研究二的结果可归纳为以下几点核心发现。

第一，策略干预对AI图像识别能力具有显著且稳健的促进效应。无论以整体正确率还是信号检测理论的敏感度指标*d'*衡量，实验组均显著优于对照组，效应量达到中等水平（_g_ ≈ 0.49），在控制人口统计学变量和AI素养协变量后结论保持一致。

第二，干预效果体现为整体辨别力的均衡提升，而非系统性判断偏移。实验组对AI图像和真实图像的识别准确率均有所提升，判断标准_c_无显著变化，排除了过度怀疑假设。

第三，干预效果受前测自评辨别能力的显著负向调节：基线自评能力越低的参与者从干预中获益越多，提示策略训练对认知起点较低的群体具有更强的赋能价值。

第四，AI使用频率不构成干预效果的调节变量，使用频次与策略习得属于不同的能力提升路径。

第五，在不可反向搜索的图像上，干预效果尤为突出（_g_ = 0.522），进一步凸显了内化辨别策略在无外部信息辅助情境下的实用价值。
